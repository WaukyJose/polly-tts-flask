# Polly TTS Flask App

A simple open-source Flask app for generating speech using Amazon Polly.

## Setup

```bash
git clone https://github.com/WaukyJose/polly-tts-flask.git
cd polly-tts-flask
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

AWS Credentials

Configure with:
aws configure

RUN
python app.py
